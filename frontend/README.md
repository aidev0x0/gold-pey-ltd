# Financial Analyst Assessment - Frontend

Angular frontend application for the Financial Analyst Assessment platform.

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Development

Run the development server:

```bash
npm start
```

The application will be available at `http://localhost:4200`

## Build

Build for production:

```bash
npm run build
```

## Configuration

Update the API URL in `src/app/services/assessment.service.ts` if your backend is running on a different port.
